Principles of Web Design

  1. Good web design is

    Simple

    Keep it simple, stupid. The mother of all design principles still applies to the web.

  2. Good web design is

    Familiar

    The web is built on patterns. Rely on familiarity and avoid breaking expectations. Don't reinvent the wheel. Focus on the design problems unique to your product and avoid superfluous design innovation.

  3. Good web design is

    Tested

    Data is a powerful tool in web design, but just like any tool, it can be abused. Gather and utilize qualitative and quantitative data and constantly strive to improve processes.

  4. Good web design is

    As little design as possible

    And it's as little code as possible. Humans and machines alike will always understand the simpler solution.

  5. Good web design is

    Iterative

    The web is a malleable platform, and web design is never finished. Constantly strive to make things better.

  6. Good web design is

    Unobtrusive

    The web is about information. Make it easy to find and stay out of the way.

  7. Good web design is

    Fast

    The web is about the speed of information. Strive to make things faster. Don't slow things down.
